# Colour

The colour palette comprises **colour tokens** and **global colours**.

Colour tokens (e.g. `text.primary`) are organised by the context in which they may be used. Where possible, prefer using colour tokens.

Global colours (e.g. `neutral.7`) do not specify a context , and should be used only if an appropriate colour token is not available.

## Example

```tsx
import { text, background } from '@guardian/source-foundations';

const headline = css`
	color: ${text.primary};
	background: ${background.primary};
`;

/*
.class-name {
  color: #121212;
  background-color: #ffffff;
}
*/
```

## Colour tokens

Colour tokens are grouped by theme:

-   [Default](https://theguardian.design/2a1e5182b/p/1377a6-tokens/b/293ddb)
-   [Core Brand](https://theguardian.design/2a1e5182b/p/1377a6-tokens/b/4893db)
-   [Alternative Brand](https://theguardian.design/2a1e5182b/p/1377a6-tokens/b/21c6cc)

The following categories of colour tokens are available:

-   Text
-   Background
-   Border
-   Line
