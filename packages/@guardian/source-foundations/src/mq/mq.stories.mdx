# Media queries

The media queries API provides `max-width` and `min-width` media queries using a
plain English syntax. The resulting queries map to breakpoints defined in our
[grid layout](https://theguardian.design/2a1e5182b/p/41be19-grids).

## Example

```tsx
import { from, until, between } from '@guardian/source-foundations';

const styles = css`
	padding: 0 10px;

	${from.mobileLandscape} {
		padding: 0 20px;
	}

	${between.phablet.and.desktop} {
		padding: 0 32px;
	}

	${until.wide} {
		padding: 0 40px;
	}
`;
```

You can also use the [object style syntax](https://emotion.sh/docs/object-styles):

```tsx
import { from, until, between } from '@guardian/source-foundations';

const styles = css({
	padding: '0 10px',
	[from.mobileLandscape]: {
		padding: '0 20px',
	},
	[between.phablet.and.desktop]: {
		padding: '0 32px',
	},
	[until.wide]: {
		padding: '0 40px',
	},
});
```

## Breakpoints

-   `wide`: 1300px
-   `leftCol`: 1140px
-   `desktop`: 980px
-   `tablet`: 740px
-   `phablet`: 660px
-   `mobileLandscape`: 480px
-   `mobileMedium`: 375px
-   `mobile`: 320px

## API

### `from`

e.g. `from.tablet` returns `"@media (min-width: 740px)"`

### `until`

e.g. `until.desktop` returns `"@media (max-width: 979px)"`

### `between...and...`

e.g. `between.tablet.and.leftCol` returns `"@media (min-width: 740px) and (max-width: 1139px)"`
